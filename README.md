# 📅 방학 시간표 관리

방학 계획과 하루 시간표를 관리하는 웹앱입니다. 요일별 시간표 틀을 만들어 두고, 날짜별로
실천을 체크하면 실천율이 자동으로 집계됩니다. 기록은 브라우저에 저장됩니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/vacation-schedule/

## 주요 기능
- **방학 기간 설정** · 기간 달력 보기
- **요일별 시간표 틀** — 요일마다 반복되는 일정 등록
- **날짜별 실행 체크** · 오늘의 할 일 · 메모
- **실천율 통계** — 전체 / 요일별 / 카테고리별 자동 집계
- **데이터 관리** — 내보내기·불러오기 (브라우저 localStorage 저장)

## 사용 방법
- 위 주소를 열면 바로 사용할 수 있습니다. 설치가 필요 없습니다.
- 기록은 사용하는 브라우저에 저장되므로 같은 기기·브라우저에서 이어서 관리하세요.
